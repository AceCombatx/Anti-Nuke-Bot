Frequently Asked Questions

Common questions about Anti Nuke Bot V5

What's new in V5?

Anti Nuke Bot V5 introduces several major new features:

  • Anti-Selfbot Detection: Advanced behavioral analysis to detect automated accounts
  • Enhanced Alt Detection: Improved algorithms for identifying alternative accounts
  • Advanced Message Filtering: Comprehensive content moderation with regex support
  • Interactive Setup Wizard: Guided configuration for easy setup
  • Comprehensive Help System: Interactive troubleshooting and detailed guides
  • Enhanced Logging: More detailed security logs and analytics
  • Improved Performance: Faster detection and response times

How accurate is the selfbot detection?

Our selfbot detection system uses advanced behavioral analysis with multiple detection factors. The accuracy depends on the sensitivity level:

  • Low Sensitivity: ~95% accuracy, minimal false positives
  • Medium Sensitivity: ~90% accuracy, balanced detection
  • High Sensitivity: ~85% accuracy, catches more sophisticated selfbots

We recommend starting with medium sensitivity and adjusting based on your server's needs. Always manually verify detections before taking permanent action.

Why aren't the protection features working?

All protection features require proper setup before activation. Use our interactive troubleshooting:

  1. Run /setup-wizard for guided configuration
  2. Ensure logging is set up: /setup-logging #channel
  3. Add trusted admins: /whitelist add @Admin
  4. Enable features individually with their config commands
  5. Verify setup: /protection-status
  6. For specific issues: /errors

What permissions does the bot need?

For optimal protection, Anti Nuke Bot V5 requires Administrator permissions. This ensures:

  • Complete access to monitor all server activities
  • Ability to reverse malicious actions instantly
  • Permission to manage roles, channels, and members
  • Access to audit logs for comprehensive monitoring
  • No permission conflicts that could compromise protection

Important: The bot's role must be positioned at the top of your role hierarchy to manage all other roles effectively.

How do I prevent false positives?

To minimize false positives:

  • Whitelist trusted staff: Add all administrators and moderators
  • Adjust sensitivity: Start with lower sensitivity and increase gradually
  • Configure thresholds: Set appropriate limits for your server size
  • Use warning mode: Start with warnings before implementing punishments
  • Monitor logs: Review detection logs to identify patterns
  • Regular review: Periodically review and adjust settings

Can I customize the detection algorithms?

Yes! V5 offers extensive customization options:

  • Sensitivity Levels: Adjust detection sensitivity for each module
  • Custom Thresholds: Set specific limits for your server's needs
  • Punishment Escalation: Configure progressive punishment systems
  • Time Windows: Adjust monitoring periods for different activities
  • Whitelist Rules: Create complex whitelist configurations
  • Custom Filters: Add server-specific content filters

Use the respective config commands or /setup-wizard for guided customization.

How do I get help if something isn't working?

V5 includes comprehensive help resources:

  • Interactive Troubleshooting: /errors for step-by-step problem solving
  • Feature-Specific Help: Commands like /nuke-help, /spam-help, etc.
  • Setup Assistance: /setup-wizard for guided configuration
  • Status Checking: /protection-status to identify issues
  • Support Server: Join our Discord for live assistance
  • Documentation: Comprehensive guides and tutorials

Is Anti Nuke Bot V5 free?

Yes! All core security features in Anti Nuke Bot V5 are completely free, including:

  • All protection modules (Anti-Nuke, Anti-Raid, Anti-Spam, etc.)
  • Advanced detection algorithms
  • Comprehensive logging and monitoring
  • Interactive help and troubleshooting
  • Configuration backup and restore
  • 24/7 support access

We believe server security should be accessible to everyone. Optional premium features may be available in the future to support development, but core protection will always remain free.
